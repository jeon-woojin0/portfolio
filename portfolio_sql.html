프로젝트 개요

추천 시스템, SQL 만으로 가능한가?

Python ML 라이브러리 없이 순수 BigQuery SQL로 5종의 추천 시스템을 구현했습니다. Bayesian Average로 리뷰 수 편향을 보정하고, UNNEST와 윈도우 함수로 카테고리 연관 구매(협업 필터링 개념)를 SQL 단에서 구현했습니다.

핵심 분석 결과

5종 추천 시스템 상세

각 시스템은 독립적으로 동작하며, 실제 서비스 상황(메인 페이지, 검색, 상품 상세 등)에 맞게 설계했습니다.

System 01 · 메인 페이지

Bayesian Average 기반 인기 상품 추천

단순 평점 평균은 리뷰 1개짜리 5.0점 상품이 상위에 올라오는 편향 발생. Bayesian Average(전역 평점 × 전역 리뷰 수 가중 보정)로 리뷰 수와 평점을 동시에 반영한 신뢰 가능한 순위를 산출했습니다.

System 02 · 카테고리 페이지

카테고리별 베스트셀러 추천

PARTITION BY 카테고리 + ROW_NUMBER 윈도우 함수로 각 카테고리 내 Bayesian 점수 상위 N개를 추출. 카테고리 이동 없이 해당 섹션 내 최적 상품을 노출하는 로직.

System 03 · 검색 연동

텍스트 키워드 필터링 추천

상품명·설명 컬럼을 LIKE + LOWER()로 키워드 매칭. UNNEST(SPLIT())로 다중 카테고리 태그를 행으로 분리해 교집합 매칭까지 구현. 검색어와 관련도 높은 상품을 평점 순으로 정렬해 반환합니다.

System 04 · 프로모션

할인율 기반 특가 상품 추천

discount_percentage 컬럼 파싱(문자열 → 숫자) + 최소 리뷰 수 임계값 필터로 "진짜 인기 있는데 많이 깎인" 상품을 선별. 단순 최저가 목록과 차별화된 큐레이션.

System 05 · 상품 상세 페이지

카테고리 연관 구매 (협업 필터링 개념)

같은 상위 카테고리 내 구매된 상품 조합을 CTE로 집계해 공동 구매 빈도 기준으로 연관 상품 추출. 주문 단위 데이터 없이 카테고리 근접성으로 협업 필터링 개념을 SQL로 근사했습니다.

핵심 인사이트

Bayesian Average 적용 시 리뷰 1~2개짜리 5.0점 상품이 상위권에서 완전히 제거됩니다. 동일 데이터에서 단순 평점 기준과 Bayesian 기준의 Top 10이 70% 이상 달라졌습니다.

SQL 구현 하이라이트

Bayesian Average SQL 로직

System 01의 핵심 로직입니다. 전역 평균과 전역 리뷰 수를 CTE로 먼저 계산하고 상품별 보정값을 적용합니다.

WITH global_stats AS (
  SELECT
    AVG(rating)              AS global_avg_rating,
    AVG(rating_count)        AS global_avg_count
  FROM amazon_products
  WHERE rating IS NOT NULL
    AND rating_count IS NOT NULL
),

bayesian AS (
  SELECT
    p.product_id,
    p.product_name,
    p.category,
    p.rating,
    p.rating_count,
    p.discounted_price,
    -- Bayesian Average 공식: (C × m + n × R) / (C + n)
    ROUND(
      (g.global_avg_count * g.global_avg_rating
        + p.rating_count * p.rating)
      / (g.global_avg_count + p.rating_count),
      2
    ) AS bayesian_score
  FROM amazon_products p
  CROSS JOIN global_stats g
  WHERE p.rating_count >= 50  -- 최소 리뷰 수 임계값
)

SELECT *
FROM bayesian
ORDER BY bayesian_score DESC
LIMIT 20;

C = 전역 평균 리뷰 수, m = 전역 평균 평점, n = 상품 리뷰 수, R = 상품 평점. CROSS JOIN으로 전역 통계를 모든 행에 조인해 단일 쿼리로 처리합니다.
